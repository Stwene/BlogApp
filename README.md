# Blog App (Full)

This archive contains a full-stack blog application starter:

- Backend: Node.js + Express + MongoDB (Mongoose)
- Frontend: React + Vite

## Quickstart

1. Backend:
   - cd server
   - npm install
   - copy .env.example -> .env and set MONGO_URI and JWT_SECRET
   - npm run dev

2. Frontend:
   - cd client
   - npm install
   - npm run dev
